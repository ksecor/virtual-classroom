# Commons Virtual Classroom

A browser-based multiplayer classroom built with A-Frame, Express, and Socket.io.

## Run it

```bash
npm install
npm start
```

Then open [http://localhost:3000](http://localhost:3000) in two browser windows to see multiplayer movement and chat.

## Controls

- Enter a display name to join the room.
- Move with `W`, `A`, `S`, and `D`.
- Click and drag to look around.
- Use the chat panel to speak with everyone in the room.
